# FindMyDojo - Martial Arts DOJO Directory

**FindMyDojo** is a full-featured web platform that helps martial arts students discover, review, and connect with dojos worldwide. Dojo owners can create listings, manage events, and engage with their community.

## Live Website


## Features
- Public browsing of verified dojos with search and filters
- Detailed dojo pages with images, schedule, pricing, reviews, and upcoming events
- Student accounts: write reviews with star ratings
- Dojo owner dashboard: add/edit dojos, create events
- Admin panel: approve dojos, view messages
- Secure authentication, CSRF protection, password hashing
- Database-driven with MySQL (PHP + PDO)

## Technologies Used
- PHP
- MySQL (XAMPP)
- HTML5, CSS3 (custom design with glassmorphism and gradients)
- JavaScript (vanilla + AOS animations)
- Font Awesome icons

## Setup (Local Development)
1. Import `dojo.sql` into your database
2. Create `.env` file with your DB credentials (see config.php)
3. Run on XAMPP / local PHP server
